# Steam-monitor-vk
Скрипт для мониторинга скорости загрузки игр в Steam.

## Описание
Решение тестового задания для стажировки в VK. Скрипт отслеживает скорость загрузки, статус и прогресс загрузки игр в Steam.

## Функции
- Определение пути установки Steam (Windows/macOS/Linux)
- Мониторинг скорости загрузки каждую минуту
- Отображение статуса (загрузка/пауза)
- Прогресс-бар и расчет оставшегося времени
- Симуляция данных для демонстрации

## Использование
```bash
python steam_monitor.py
